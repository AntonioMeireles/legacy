What's new

Foresight 2.0.2 is a minor point release featuring 

GNOME Display Manager

The GNOME Display Manager (GDM) has been updated and features an all new interface when logging in to your desktop.  Users are presented with a list of users, and clicking on the user will bring up the password dialog.  (Users can still type their user name and GDM will now auto-complete the username as well).  Language selection has been moved to the bottom center of the screen.

New Kernel

Foresight's kernel has been updated to 2.6.25, featuring better hardware compatibility.  Improved support for Intel's i915 and ATI r500 graphics, new sound card drivers, and better wireless support, including Atheros.  For a full list of kernel changes see: http://kernelnewbies.org/LinuxChanges

This new kernel also fixes issues with the Foresight Live Media.  New Live Media including LiveCDs and virtual images will be available in the near future.

NetworkManager

NetworkManager has been updated to include much better stability, specifically for users with a wired connection.

Mono Stack

The Mono stack has been re-built to be faster and improve memory consumptions.  Users using Banshee, F-Spot and other Mono applications will benefit from the improved Mono stack.

Banshee 1.0 Preview

Foresight features Banshee 0.13.2 as the default music manager.  Banshee's upcoming 1.0 release is also available for testing, and can be installed side by side with Banshee 0.13.2.  Banshee 1.0 (0.99.2) features the addition of video playback, improved Last.fm integration, burning integration with Brasero, improved Podcast support, and an all new interface.  To install, search PackageKit for "banshee-1" and install.

Filing Issues and Bugs

Please report any issues or bugs you encounter while using Foresight 2.0.2.  Foresight's issue tracker, JIRA, is available at http://issues.foresightlinux.org.

Getting Help

Help is available in many forms, and you can choose what you're most comfortable with.

    * IRC:  Visit the Foresight IRC channel, #foresight on Freenode, and ask questions.  We have one of the most friendly IRC channels you'll come across with everyone from users to developers reaching out to help answer questions.
    * Forums: Our forums continue to grow, and are a good source of information to check if a specific problem or question has come up before.
    * Wiki: Documentation on the wiki is growing on a daily basis, with updates often to the Frequently Asked Questions and other how-to's to get you going with Foresight Linux.

Contribute to Foresight Linux

Foresight Linux has a vibrant and friendly community, ready to assist any level of user.  Participation is also welcome and encouraged!  There are several different ways to engage with the Foresight community:

* You can find us in #foresight on chat.freenode.net.  Foresight users can click the X-Chat GNOME icon in Applications -> Internet menu to access IRC.

* Contribute to the Foresight Wiki.

* Package and maintain software.  Visit the Developer Wiki at http://wiki.foresightlinux.com/confluence/display/DEV/Foresight+Linux+Development for more information.
